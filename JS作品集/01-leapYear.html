判斷是否為閏年？

判斷方式：(取自維基百科)
1. 公元年分除以4不可整除，為平年。
2. 公元年分除以4可整除但除以100不可整除，為閏年。
3. 公元年分除以100可整除但除以400不可整除，為平年。
4. 公元年分除以400可整除，為閏年。

程式碼 
        my$('btn').addEventListener("click", function () {
        var year = my$('ip').value;
        var dv = my$('dv');
        if(year && year.length == 4){
            year % 4 == 0 && year % 100 != 0 || year % 400 == 0 ? dv.innerText = year + "為閏年" : dv.innerText = year + "為平年";
        }else {
            dv.innerText = "請輸入正確的年份(4位數)";
        }
    }, false)
